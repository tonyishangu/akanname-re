Akan Name Generator

This is a web application that uses a user's birthday to calculate the day they were born then gives their akan name based on gender selcted.Akan names belong to a Ghanian culture, children are given names depending on the day they were born.

The table below shows a list of Akan names. If you can't quite recall the day of the week you were born, consider using the name generator.

  Male Female
Sunday Kwasi Akosua
Monday Kwadwo Adwoa
Tuesday Kwabena Abenaa
Wednesday Kwaku Akua
Thursday Yaw Yaa
Friday Kofi Afua
Saturday Kwame Ama
